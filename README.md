# KC_Production_Website

Modern marketing website for **K&C Design & Manufacturing** — precision CNC machining, fixtures, tooling, and production manufacturing.

## Stack

- React + Vite
- Tailwind CSS
- React Router
- lucide-react

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Built files output to `dist/`.
